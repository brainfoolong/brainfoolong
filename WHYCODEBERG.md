### Why i move my repositories to Codeberg?

* Codeberg e.V. is a registered non-profit association based in Berlin, Germany. I live in Austria, so it's natural fit to host my stuff on proper non-profit hosts. 
* GitHub felt unreliable in the last years and the shift the platform makes doesn't fit my mind anymore
* Having sources spread across multiple services feels better, development still only is done on Codeberg Repos.
* Codeberg forces me to host my own action runners, which is a nice thing once you have it. Full control for CI pipeline.

### Why i still have Github repositories?
* Mirror - Having multiple sources for open-source software is always a good thing
* I started almost every repo i currently have on Github, so the audience is mainly still on GitHub


That's it, nothing spectacular.
